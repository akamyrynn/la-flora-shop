## Как запустить прямо сейчас

1. ✅ Supabase настроен (ID и ключ добавлены)
2. ✅ Схема базы готова (`supabase-products-schema.sql`)
3. ✅ Примеры данных готовы (`sample-data.sql`)
4. ✅ Сервер запущен (`npm run dev`)

### Следующие шаги:

1. Выполните SQL из `supabase-products-schema.sql` в Supabase SQL Editor
2. Выполните SQL из `sample-data.sql` для тестовых данных
3. Откройте http://localhost:3000
4. Перейдите на `/admin` и добавьте свои букеты
5. Загрузите изображения в Supabase Storage или используйте внешние URL
