# Быстрый старт

## 1. Выполните SQL в Supabase

Зайдите в SQL Editor вашего проекта Supabase и выполните:

```sql
-- Скопируйте и выполните содержимое файла supabase-products-schema.sql
```

Это создаст таблицы:
- `products` - букеты
- `addons` - допы (открытки, вазы, кризал)
- `reviews` - отзывы

## 2. Установите зависимости

```bash
npm install
```

## 3. Запустите проект

```bash
npm run dev
```

Откройте http://localhost:3000

## 4. Добавьте первый букет

1. Перейдите на `/admin`
2. Нажмите "Добавить букет"
3. Заполните форму:
   - Название: "Букет из 15 красных роз"
   - Цена: 2500
   - Тип: Монобукет
   - Вид цветка: Розы
   - Цвет: Красный
   - Количество: 15
   - Добавьте URL изображения
4. Сохраните

## 5. Добавьте допы

1. Перейдите на вкладку "Допы"
2. Добавьте:
   - Открытка (150₽)
   - Ваза (500₽)
   - Кризал (100₽)

## Готово!

Теперь можно:
- Просматривать каталог на `/catalog`
- Фильтровать букеты
- Добавлять в корзину
- Оформлять заказы

## Связь вариантов (15/25/51/101 роза)

Чтобы связать варианты букетов:

1. Создайте первый букет (15 роз)
2. Скопируйте его `id` из таблицы
3. Создайте остальные варианты (25, 51, 101)
4. В поле `variant_group` укажите один и тот же UUID для всех вариантов
5. Теперь на странице товара будут кнопки переключения между вариантами

## Загрузка изображений

Для загрузки изображений:

1. Создайте bucket в Supabase Storage (например, "products")
2. Загрузите изображения
3. Получите публичный URL
4. Вставьте URL в форму добавления товара

Или используйте внешние URL (например, с вашего CDN)
